Primeiro Paragrafo

git Init: Cria o repositório no diretório atual; Git Status: Verifica o Status do repositório; Git Add (nome do arquivo ou .): Adiciona o arquivo para o commit. O "." é para todos os arquivos do diretório; git commit - m "Nome do Commit": Faz o commit com a descrição do commit; git push endereço master: Envia a versão para a ferramenta de controle de versão; git pull endereço master: Traz os arquivos do repositório. O Repositório precisa existir no diretório; git clone endereço master: Traz os arquivos do repositório que não existe no diretório.
